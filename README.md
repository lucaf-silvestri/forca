# Jogo da Forca
Um jogo da forca simples feito em Python.

## Como Jogar
1. Execute o script Python.
2. Tente adivinhar a palavra secreta digitando letras.
3. Você tem um número limitado de tentativas antes de perder.

## Requisitos
- Python 3.11.9 instalado

## Como Executar
`python forca.py`
